# 競輪 勝ちパターン管理（URL配布＆PWA）
このフォルダをそのまま GitHub Pages 等で公開すると、URL配布＆ホーム画面追加（アプリ風）ができます。

## GitHub Pages（無料）で公開
1) GitHubで新規リポジトリ作成（Public推奨）
2) このZIPを解凍して、中身をリポジトリ直下へアップロード
3) Settings → Pages → Deploy from a branch → main / (root) を選択
4) 表示されるURLを選手に配布

## iPhone（Safari）でアプリ風にする
SafariでURLを開く → 共有ボタン →「ホーム画面に追加」
